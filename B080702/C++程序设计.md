### 函数的重载

使一个函数名具有多种功能，称为函数重载。

```c++
int max(int, int);
int max(int, int, int);
```

### 地址运算符&
```c++
cout << &x
```



### 引用&

```c++
void main() {
  int x = 56;
  int& a = x;
  int& r = a;
  
  r = 25; // x = 25, a = 25, r = 25;
}
```
